This repository includes codes for our paper entiled "AI-Based Whole-Liver Modeling for Preoperatively Predicting Recurrence Risk in Hepatocellular Carcinoma", which proposes a fully automated survival prediction (FASP) system that mines whole-liver imaging data to preoperatively predict recurrence-free survival (RFS) in HCC patients treated with curative resection.
